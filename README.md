# Nicho — Releases

Distribuição pública do **Nicho** (o código-fonte é privado).

- **Baixar:** veja a aba [Releases](../../releases) e pegue o `.dmg` mais recente.
- **Auto-update:** o app se atualiza sozinho via Sparkle, lendo o `appcast.xml` deste repositório.

Instalação: abra o DMG, arraste o **Nicho** para Aplicativos. Na primeira abertura, clique com o botão direito → **Abrir** (o app ainda não é notarizado pela Apple).

Nicho é baseado no [ComfyNotch](https://github.com/AryanRogye/ComfyNotch), de Aryan Rogye (MIT).
